## 模型释义

|   集合类型   |     用途     |           备注           |
| :----------: | :----------: | :----------------------: |
|     User     | 外部注册用户 |        非后台用户        |
|   Category   |   内容分类   |  可关联于多种内容型数据  |
| Organization |   外部组织   | 公司、开源社区、公益组织 |
|   Activity   |     活动     |     各类小中大型活动     |
|   Program    |   活动环节   |    演讲、工作坊、展位    |
| PartnerShip  |   合作关系   |      活动合作、赞助      |
